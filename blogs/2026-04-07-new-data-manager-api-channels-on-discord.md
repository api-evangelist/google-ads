<p>
We're excited to announce the addition of new channels for the <a href="https://developers.google.com/data-manager/api?utm_source=googleblog-adsdevs&amp;utm_medium=blog&amp;utm_campaign=adr-26h1dmapi&amp;utm_content=discordintro-blogpost">Data Manager API</a> on our “Google Advertising and Measurement Community” Discord server. To join, just click <a href="https://goo.gle/dm-api-discord">this invite link</a> and follow the onboarding guide.
</p>
<p>
Our team will be on this server regularly to interact with the Data Manager API community, answering questions and gathering feedback in the <code>#data-manager-api</code> channel. We’ll also keep you up to date on the latest Data Manager API news in the <code>#announce-data-manager-api</code> channel.
</p>
<p>
If you have any questions or want to discuss this post, please reach out to us on the <a href="https://goo.gle/dm-api-discord">“Google Advertising and Measurement Community” Discord server</a>. We look forward to seeing you there and hearing your thoughts on how we can continue to improve the Data Manager API!
</p>
